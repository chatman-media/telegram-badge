API Documentation

Endpoint

GET https://telegram-badge.vercel.app/api/telegram-badge

Parameters

  • channelId - Telegram chat ID or username (e.g., @your_channel)
  • style - Badge style: flat, plastic, flat-square, for-the-badge, social
  • label - Custom label text (default: "Telegram")
  • color - Badge color in hex without # (default: "2AABEE")
  • labelColor - Label color in hex without # (default: "555555")
  • logo - Show Telegram logo: true/false (default: true)
